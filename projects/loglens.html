LogLens

Real-time error logging and monitoring platform with a live dashboard, Python SDK, and SSE-powered instant updates.

FastAPI Next.js PostgreSQL SSE Python SDK Bun

Overview

LogLens is a developer-focused error logging and monitoring platform. It provides a clean ingest API, a real-time dashboard with Server-Sent Events, and a Python SDK for one-line integration into any application.

Key Features

  • Live dashboard with SSE-powered instant error updates — no polling
  • Severity filtering (info, warning, error, critical) with color-coded indicators
  • Full-text search across log messages and stack traces
  • Time-series charts for error trend visualization
  • Python SDK for one-line integration into existing applications
  • RESTful ingest API with API key authentication

Architecture

The backend is built with FastAPI and PostgreSQL for high-throughput log ingestion. The frontend uses Next.js with SSE connections for real-time updates. The Python SDK wraps the ingest API for seamless integration.
