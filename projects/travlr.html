Travlr Getaways

Full-stack travel booking + admin management web application.

Angular Node/Express MongoDB RBAC/Auth

What it does

Travlr Getaways provides a user-facing experience for browsing trips and an admin experience for managing trip data. The system uses an API layer to connect the UI to MongoDB-backed data.

My capstone enhancements

  • Improved security/auth workflow (ex: role-based access control for admin actions).
  • Cleaned up data flow between UI → API → database to reduce errors and improve reliability.
  • Strengthened maintainability with clearer structure and better separation of concerns.
