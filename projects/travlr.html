Travlr Getaways

Full-stack travel booking and admin management platform with a structured REST API, MongoDB-backed data, and role-based access control.

Angular Node / Express MongoDB RBAC / Auth

Overview

Travlr Getaways provides a user-facing experience for browsing trips and an admin experience for managing trip data. The system uses an API layer to connect the UI to MongoDB-backed data.

Capstone Enhancements

  • Improved security/auth workflow with role-based access control for admin actions
  • Cleaned up data flow between UI, API, and database to reduce errors and improve reliability
  • Strengthened maintainability with clearer structure and better separation of concerns
