AgentForge

AI workflow agent platform for building, configuring, and running multi-step AI agents with tool use through a visual web interface.

Next.js 14 FastAPI LangChain OpenAI Supabase Python TypeScript

Overview

AgentForge is an agent orchestration platform — not a chatbot. Users create custom AI agents that chain together multiple LLM calls, use tools (web search, document parsing, code execution, data extraction), and follow multi-step workflows to automate complex business tasks. Each agent streams its execution step-by-step in real-time via SSE.

Key Features

  • Agent builder with system prompt, tool selection, and multi-step workflow editor
  • Five integrated tools: web search (SerpAPI), document reader (PDF/DOCX), sandboxed code executor, structured data extractor, and summarizer
  • Real-time SSE streaming with step-by-step execution logs
  • LangChain agent executor with OpenAI tool-calling and accumulated context across steps
  • Four pre-built agent templates: Document Analyzer, Research Agent, Data Extractor, Code Reviewer
  • Dashboard with usage stats, run history, token tracking, and API key generation
  • Supabase Auth (email + GitHub OAuth), rate limiting (10 runs/hour), and row-level security
  • CI/CD with GitHub Actions, Docker support, deployed on Vercel + Render

Architecture

The frontend is a Next.js 14 app with shadcn/ui components, consuming a FastAPI backend that orchestrates agents via LangChain. The agent executor resolves tool names to LangChain tool instances, chains workflow steps with accumulated context, and supports both tool-augmented (OpenAI function calling) and direct LLM execution. Supabase provides auth and PostgreSQL storage with RLS policies. The backend streams execution events via SSE to the frontend runner panel.
